安装：

npm install -g nodeppt

启动：

nodeppt start -d <目录>
nodeppt start -c socket
nodeppt start -d <目录> -p <端口> -H <IP> -c socket

手机控制：

按Q键

md同级可以创建img、js、css文件夹来引用静态资源，没有找到使用默认的assets文件夹下的资源

使用画笔

按下键盘【P】键：按下鼠标左键，在此处乱花下看看效果。
按下键盘【B/Y/R/G/M】：更换颜色，按下【1~4】：更换粗细
按下键盘【C】键：清空画板

宽度不够

按下键盘【W】键，切换到更宽的页面看效果，第二次按键返回

使用overview模式

按下键盘【O】键
在overview模式下，方向键下一页，【enter】键进入选中页
或者按下键盘【O】键，退出overview模式
